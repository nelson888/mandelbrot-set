# Mandelbrot set
A simple implementation of the Mandelbrot set

## How to compile
To compile, run (Linux only)
```shell script
./gradlew uberjar
```

To execute, run
```shell script
java -jar build/libs/mandelbrot.jar
```
It will open a new window. To zoom, click the point you
want to zoom
